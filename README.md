# FAP-Grade-Tool
Sửa điểm FAP, Auto tính %, lưu session chống F5


-> Nễu bạn là sinh viên FPT, bạn bị phụ huynh dí coi bảng điểm, mà kì rồi bạn lỡ dính bầu (thai lị).

## ->> Tool này dành cho bạn, sử dụng cơ chế extension và lưu lại trạng thái để bạn F5 (reload) thoải mái.
